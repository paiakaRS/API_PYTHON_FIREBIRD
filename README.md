# Firebird API Bridge (FastAPI + Pandas)

Este projeto demonstra a criação de uma camada de API moderna para integração com bancos de dados legados **Firebird**.

## 🚀 Tecnologias Utilizadas
- **Python 3.x**
- **FastAPI**: Criação de endpoints de alta performance.
- **Pandas**: Manipulação e filtragem de dados em memória.
- **SQLAlchemy**: ORM e conexão com o banco Firebird.
- **Uvicorn**: Servidor ASGI.

## 🛠️ Funcionalidades
- [x] Conexão segura com banco de dados Firebird.
- [x] Autenticação via **API Key (Token)** nos headers.
- [x] Endpoint de listagem de produtos com Pandas.
- [x] Endpoint de busca por código.
- [x] Endpoint de atualização de preços (Update) com transação SQL.
- [x] Documentação automática via **Swagger UI**.
